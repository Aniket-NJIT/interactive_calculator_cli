# Python Calculator CLI

A robust, command-line calculator application built with Python. This project demonstrates best practices in software development, including the DRY principle, comprehensive testing with `pytest`, and Continuous Integration (CI) using GitHub Actions.

## Features

- **REPL Interface:** Continuous interaction loop until the user chooses to exit.
- **Arithmetic Operations:** Supports Addition, Subtraction, Multiplication, and Division.
- **Robust Error Handling:** Gracefully handles division by zero and invalid inputs.
- **100% Test Coverage:** Verified by automated testing pipelines.

## Prerequisites

- Python 3.10 or higher
- `pip` (Python package installer)
